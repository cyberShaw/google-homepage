Google Homepage
======================
A static reconstruction of the [Google](www.google.com) Homepage.

Getting Started
----------------------
### To Run Locally:
+ Clone repository. 
+ Open index.html in a browser of your choice.

### Github Pages
+ Visit this repository's [website](https://thewhitewolf17.github.io/google-homepage/)
